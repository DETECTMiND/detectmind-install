Install DETECTMiND

Follow these steps on your Android phone. The whole process takes about 5 minutes.

Requirements

  • Android phone running Android 8.0 (Oreo) or later
  • About 30 MB of free storage
  • Your 6-character participant code from the research team

Download the app

Latest version: 1.0.0

⬇ Download DETECTMiND.apk

Download from your Android phone's browser. The file will save to your Downloads folder.

You will see security warnings. This is expected because the app is distributed directly (not via Google Play). The warnings are part of Android's standard protection — they don't mean anything is wrong. Tap through them when prompted.

Step-by-step

  1. Tap the download button above Your browser will download DETECTMiND.apk. A "this type of file can harm your device" warning may appear — tap Download anyway or Keep.
  2. Open the downloaded file Tap the download notification, or open your Files app → Downloads and tap DETECTMiND.apk.
  3. Allow installs from this source (first time only) Android will say something like "Your phone doesn't allow installation of unknown apps from this source." Tap Settings → enable Allow from this source → go back.
  4. Install Tap Install. The install takes a few seconds. Once it's done, tap Open.
  5. Read & accept the consent screen The app shows exactly what data will be collected. Take your time reading it. Tap I Agree only if you're happy to participate.
  6. Grant permissions You'll be guided through enabling each permission the app needs. Some require opening your phone's Settings app; the app provides direct shortcuts for each one.
  7. Enter your participant code Type the 6-character code your researcher sent you. Tap Enroll. You're done — the app will run quietly in the background.

If something goes wrong

"App not installed" error

You may already have an older version. Uninstall any existing DETECTMiND, then try again.

Banking app won't open while DETECTMiND is running

This is intentional Android security in your banking app — not a DETECTMiND issue. DETECTMiND will detect this and show you a notification with a one-tap shortcut to temporarily pause gesture collection. After your banking session, re-enable it via the notification.

Permission keeps being denied

Some permissions are restricted by Android for sideloaded apps. If a sensor isn't working, contact the research team — we can help you grant it via a different route.

Need to remove the app

Long-press the DETECTMiND icon → Uninstall. All locally stored data is removed. To also remove your data from the server, please contact the researcher.
